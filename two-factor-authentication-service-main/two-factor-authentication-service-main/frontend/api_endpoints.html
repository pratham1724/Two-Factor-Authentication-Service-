Api Endpoints
# HTTP method URI path Description
1 GET /authentication Retrieves all user's and their key's from database.
2 GET /authentication/generateKey Generate and return unique key for user.
3 POST /authentication/registerService Store user's {email : key} pair in the database.
4 POST /authentication/generateToken Generate and return six digit TOTP from user's key.
5 POST /authentication/verifyToken Verify user entered token form token generated from the key stored in database. Return true if Token is correct, else return false.
All endpoints can be accessed on this URL https://two-factor-authentication-api.azurewebsites.net.
